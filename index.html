nCopula

Construct and use hierarchical Archimedean copulas through compound distributions

Status

In construction

Abstract

The nCopula package aims to simplify the construction process and usage of hierarchical Archimedean copulas through compound distributions in R. It is possible to build structures with clear representations, obtain expressions for Archimedean copulas, wether they are hierarchical or not, as well as other important functions (i.e. Laplace Stieltjes Transform, pgf, etc.), given a certain path and structure. Furthermore, the generation of random vectors is possible from any given structure.

Alternative

The copula package is a very efficient alternative to our package. While it does not provide tools to construct HAC through compound distributions, it contains more stable and complete functions and classes for Archimedean copulas. In fact, we used a construction process similar to the one used in the copula package, which makes it easier for users to go from one to another. However, our packages implements various useful/flexible elements. For example, we provide expressions for the derivatives of important functions (Laplace-Stieltjes Tranform, probability generating function, etc.) of different probability distributions. This implies, in some cases, that we can provide expressions for the density of Archimedean copulas, which can speed the likelihood estimation. The reason why we chose to not use the classes already in place with the copula package is to gain more freedom in what we can add.

Features

  • Construction of Archimedean copulas
  • Construction of hierarchical Archimedean copulas though compound distributions
  • Sampling from the two type of structures above
  • Access to various expressions for different distributions and (hierarchical) Archimedean copulas
